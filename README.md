# twitter-mute-buzzhuff

Twitter がやたらおすすめ（「いまどうしてる？」）してくる Buzzfeed と ハフィントンポスト を非表示にする拡張機能。
ブロックしても表示されるのは流石にウザい。
Twitterが優遇するクソメディアはどんどん非表示にする予定。
